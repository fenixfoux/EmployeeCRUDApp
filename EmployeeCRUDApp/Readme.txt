create ASP.NET Web Application(.Net Framework) 
description(project templates for creating ASP.NET applications. You can create ASP.NET Web Forms, MVC, or Web Api Applications and add many other features in ASP.NET.))


required dependencies which should be installed(right click on Dependencies - Manage NuGet Packages )

1. Microsoft.AspNetCore.Mvc.Razor.RuntimeCompilation (installed by default when you create project)
2.Microsoft.EntityFrameworkCore
3.Microsoft.EntityFrameworkCoreDesign
4.Microsoft.EntityFrameworkCoreTools
5.Microsoft.VisualStudio.Web.CodeGeneration.Design  (required for create Views with Scaffolding, if create view manually this isn't needable)
6.Oracle.EntityFrameWorkCore   (if using OracleDataBase if not use framework for yor database)
7.Oracle.ManagedDataAcces.Core (if using OracleDataBase if not use framework for yor database)
